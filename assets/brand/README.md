# Visuels Flam's

Depose ici les visuels que tu veux utiliser dans l'application :

- logos `.svg`, `.png`, `.jpg`
- icones
- fonds de carte
- motifs
- photos produit
- elements de marque

## Nommage conseille

- `logo-flams.svg`
- `logo-flams-white.svg`
- `card-background.png`
- `pattern-loyalty.svg`
- `restaurant-placeholder.jpg`

## Notes

Les fichiers dans ce dossier pourront ensuite etre references depuis le code avec un chemin comme :

```html
<img src="assets/brand/logo-flams.svg" alt="Flam's" />
```

Pour la future version Next.js, ces fichiers iront probablement dans `public/brand/`.
